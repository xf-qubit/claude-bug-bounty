# Wordlists

Curated wordlists for fuzzing and parameter discovery.

| File | Contents |
|:---|:---|
| `common.txt` | Common directories and endpoints |
| `api-endpoints.txt` | REST API endpoint patterns |
| `params.txt` | Common HTTP parameter names |
| `sensitive-files.txt` | Backup files, config leaks, .env patterns |
| `raft-medium-dirs.txt` | RAFT medium directory wordlist |
| `REFERENCES.md` | Pointers to SecLists · OneListForAll · fuzz4bounty · PayloadsAllTheThings |

For larger wordlists, see [SecLists](https://github.com/danielmiessler/SecLists).
